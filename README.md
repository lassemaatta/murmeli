# Murmeli, a Clojure wrapper for the MongoDB Java driver

Wraps version 5.1 of the MongoDB Java [driver](https://www.mongodb.com/docs/drivers/java/sync/v5.1/).

Status: alpha

## Installation

Download from https://github.com/lassemaatta/murmeli.
